# HADROS3 Figure Index

This index tracks figures produced from reproducible HADROS3 runs.

Each entry should use:

```text
Figure ID:
File:
Run ID:
Description:
Parameters:
Physical message:
Status: diagnostic / validation / paper_candidate
```

Example:

```text
Figure ID: FIG-DIS-001
File: Fig_DIS_tau_vs_energy_GBW_IIM.png
Run ID: RUN_DIS_energy_scan_YYYYMMDD_hash
Description: Optical depth growth with neutrino energy for GBW/IIM.
Parameters: E_nu scan, fixed torus density, fixed spin.
Physical message: DIS optical depth increases with UHE neutrino energy.
Status: paper_candidate
```
