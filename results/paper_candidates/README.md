# Paper Candidate Results

Only promote figures or tables here when they satisfy the paper-candidate
criteria:

```text
1. reproducible run
2. complete provenance
3. theory_version recorded
4. physical parameters documented
5. corresponding validation passed
6. caption written
7. physically interpretable result
8. no hidden proxy dependency
```

Suggested structure:

```text
figures/
tables/
captions/
```

Each promoted figure should have:

```text
PNG
PDF/SVG when possible
caption.md
source_run.json
```
